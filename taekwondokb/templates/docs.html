Документація для розробників

1. Зареєструйтесь в системі

Зареєструватися
Увійти

2. створіть API Key

Дайте назву ключа і система автоматично його згенерує

Ключ має стандарт uuid4, Наприклад: 16fd2706-8baf-433b-82eb-8c7fada847da

3. Requests

Всі запити формуються в такому форматі

https://{{ request.META.HTTP_HOST }}/api/v1/<:token>/<:endpoints>

Де token - це токен аутентифікації користувача, а endpoint лінк до данних

3. Endpoints

/encyclopedia/ -- Список всіх томів

/encyclopedia/<:tom>/ -- Зміст конкретного Тому.

:tom - Номер Тому

/encyclopedia/<:tom>/:page>/ -- Сторінка.

:tom - Номер Тому

:page - Номер сторінки

/belts/ -- Список всіх поясів

/belts/<:kupslug> -- Нормативи на конкретний пояс

:kupslug - ID поясу

Приклад:

https://{{ request.META.HTTP_HOST }}/api/v1/<:token>/belts/9-kup -- Даний запит виведе нормативи на біло-жовтий пояс в форматі JSON
